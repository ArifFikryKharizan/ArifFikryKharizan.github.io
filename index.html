Assignment

Website Performance Report

For our team's first Web Programming assignment, we were told to do some website performance analysis using Google's Lighthouse. Lighthouse to make it simple is an extension made by Google to analyze any kind of website to make sure that the website in common is user friendly and compatible with every devices. So for our assignment we have decided to make a report on the Top Newspaper Website in Malaysia.

During the assignment i was task to make a research about Berita Harian and Harian Metro which are known to be the most reliable news anchor for Malaysians. Not only i learn about the history of the reason of why the company existed, I also learn about the importance of a TRUSTWORTHY source when we want to know about the latest news around us.

To be brief, most of these news website gets a B+ in their website architecture as they are lacking some features that most website that uses HTML5 but it does not hinder the users experience to an extreme level, sure the amount of ads is very intrusive but with an adblock enable, it is a rather simple website for everyone to use.

In my opinion this assignment can a rather hit or miss for student as it was done at the early stages of our studies and seeing these ruin magic(html codes) are somehow confusing for us who are new to web programming but as we are nearing the end of this semester, the assignment did help a lot as we are now more conscious regarding our website usability and compatibility.

jobOnCampus

For our project in this subject, we have decided to build a website about making UTM student's life more easier with a dedicated jobOnCampus website that is build from the scratch-up. This website took everything that was bad about the in-build JobOnCampus(JoC) that is in UTM portal and revamp it with a more modern design with simple and easy interface for anyone to use.

This is due to the fact that JoC is not fully utilized in out university and has been an issue with the MPP, we thought just maybe we are able contribute to solve said problem. The website is still in its infant stages and as we know, babies will eventually mature into adults thus we are still tinkering it to make sure that the website is SAFE & USER-FRIENDLY for students and also staffs.

My part was to handle the LOGIN page of the website. I have created the options for students to SIGN IN if they already registered an account and SIGN UP for students who wishes to use this website. This page has been coded to store any data that user has inputted with a proper PHP and SQL database so we can technically say that the website is already on its final stages before its release
